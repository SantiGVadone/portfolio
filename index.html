Projects

Inventory Hub: Self-Hosted Full-Stack Stock System

A mobile-first inventory management application built with React Native and TypeScript, powered by a Node.js REST API and a PostgreSQL database. The entire ecosystem is fully containerized using Docker Compose and self-hosted on a dedicated Linux home server, utilizing Cloudflare Tunnels to expose endpoints securely to the web without open ports. Features real-time stock updates, modern full-stack architecture, and full infrastructure control.

GlowManager API: Hair Salon & Appointment Management Engine

A production-ready REST API built with Node.js and TypeScript to digitize salon operations. It features a layered architecture (Controller-Service-Repository) and a PostgreSQL database. The system implements Role-Based Access Control (RBAC) via JWT for Admins, Bosses, and Employees, handling appointments, client histories, service catalogs, and automated staff bonus calculations. Robust security includes password hashing and runtime schema validation using Zod.

© 2026 VadoneDeV

HomeServer Setup: Self-Hosted Infrastructure & DevOps Environment

A personal homelab infrastructure built by transforming an old laptop into a dedicated Linux server. The project covers the configuration of a secure deployment pipeline utilizing SSH remote access, version control with GitHub, and service isolation through Docker and Docker Compose. It manages the life cycle of containerized full-stack applications, handling container networking and volumes while implementing Cloudflare Tunnels to orchestrate secure external access. This project bridges the gap between software development and system administration.
